Introduction to flexbox

Flexbox is a flexible box that can be used to set up element with container

                        <div class="container-m3">
    <div>
        <h2>Flex-Container</h2>
    </div>
    <div>
        <ul 
            style="display: flex; list-style: none; padding: 0; column-gap: 15px;" 
        >
            <li class="item-m3">
                <p>Flex-Item</p>
            </li>
            <li class="item-m3">
                <p>Flex-Item</p>
            </li>
            <li class="item-m3">
                <p>Flex-Item</p>
            </li>
        </ul>
    </div>
</div>
